Hard-skills

Compétences techniques développées au fil de mes projets académiques et personnels :

  • 💻 Installation de systèmes Linux (Xubuntu, Debian)

    Installation complète à partir de zéro : partitionnement, configuration réseau, gestion des utilisateurs, mises à jour système, et installation d'outils de développement en ligne de commande.

  • 🌐 Configuration de serveurs web (Apache2)

    Déploiement de serveurs web Apache, activation de modules comme userdir, gestion des fichiers d'index et des répertoires personnels d'utilisateurs (public_html).

  • 🛠️ Développement web avec PHP, HTML, CSS

    Création d'interfaces web avec HTML/CSS, logique dynamique avec PHP, manipulation de formulaires, affichage conditionnel et gestion des erreurs côté serveur.

  • 🔗 Interaction PHP & MariaDB via PDO

    Connexion sécurisée à MariaDB avec PDO, requêtes préparées, création de bases, tables, insertion et affichage dynamique de données dans des pages web.

  • 🧬 Bases de données : SQL, PostgreSQL, MariaDB

    Écriture de requêtes SQL (SELECT, INSERT, GRANT), création d'utilisateurs, gestion des permissions et des relations entre tables sous PostgreSQL et MariaDB.

  • 🛰️ Services réseau : FTP, SSH, HTTP

    Installation et configuration de serveurs FTP (vsftpd), services SSH sécurisés, ports personnalisés, gestion des utilisateurs locaux et anonymes.

  • 📡 Analyse réseau avec Wireshark

    Capture et analyse de trames réseau, décodage des protocoles (HTTP, FTP), identification de mots de passe codés en base64, compréhension des encapsulations réseau.

  • 👨‍💻 Programmation Java orientée objet

    Modélisation UML et implémentation Java de jeux interactifs (ex : échecs), utilisation de classes abstraites, héritage, polymorphisme, logique de validation métier.

  • 📐 Modélisation UML : diagrammes de classes

    Construction de diagrammes de classes clairs avec associations, héritages, méthodes, attributs, réflexion orientée objet appliquée à des systèmes réels.

  • 🔍 Débogage PHP/Java & tests

    Activation du mode debug PHP, affichage des erreurs, tests de scripts web et programmes Java depuis le terminal avec affichage en temps réel des erreurs.

  • 🖥️ Virtualisation : VirtualBox & Marionnet

    Utilisation de machines virtuelles via VirtualBox, déploiement d'environnements Marionnet pour simuler des réseaux virtuels et exécuter des TP réseau complets.

  • 🧑‍💻 Scripts Shell & commandes Linux

    Maîtrise des scripts bash pour automatiser les tâches, gestion de fichiers, droits, utilisateurs, et interaction avec le système via le terminal.
